Conditionals

There are two ways to do conditionals in JS - if statements and ? statements.

The following examples do not handle test cases that are exactly 4.

There is also no error handling for incorrect types.

            function conditional(){
                let a = prompt("Enter a value for 'a'");

                (a > 4) ? alert("a is greater than 4") : alert("a is lesser than 4");
            }



Logical Operators

OR, AND, NOT

||, &&, !

            function noZero(){
                let a = prompt("input a value for a");
                let b = prompt("input a value for b");

                if(a == 0 || b == 0){
                    alert("you lose!")
                } else {
                    alert("you win!")
                }
            }

Don't enter zero to win!

Nullish Coalescing Operator - ??

a ?? b returns a if a is not null or undefined, otherwise it returns b.

This can be used in an instance where we want to omit falsy results and only check for empty results.

e.g. let height = 0; height ?? 100; if we used ||, we would instead return 100. ?? will return 0.

Switch Statements

Switch statements are like a chain of if statements, in a more concise syntax.

            function (){
                let a = prompt("Enter a number between 1 and 5");

                switch(a){
                    case "1": alert("ONE!!!"); break;
                    case "2": alert("TWO!!!"); break;
                    case "3": alert("THREE!!!"); break;
                    case "4": alert("FOUR!!!"); break;
                    case "5": alert("FIVE!!!"); break;
                    default: alert("Nope, that aint one of em'! Try Again!"); break;
                }
            }
